API Documentation

/process_chehra_frame - POST

Processes the provided image frame for facial emotion recognition using the Chehra model.

Method: POST

URL: /process_chehra_frame

Request Body (form-data):

{
    "frame": "[binary image data]"
}

Response:

{
    "encoded_image": "[binary image data]"
}

/get_chehra_predictions - GET

Retrieves the latest predictions made by the Chehra model.

Method: GET

URL: /get_chehra_predictions

Response:

{
    "predictions": [array of predictions]
}

/create_file/{session_id}/{model_name} - POST

Creates a file with provided content, associating it with a given session ID and model name.

Method: POST

URL: /create_file/<session_id>/<model_name>

URL Parameters:

        session_id: Unique identifier for the session.
        model_name: Name of the model.

Request Body (JSON):

{
            "content": "Your content here..."
        }

Response:

{
            "result": "File [file_name] created in folder [folder_name]"
        }
